All Books + Free PDFs
Every book used in this roadmap — with working GitHub PDF links (2–3 mirrors each)
Completely free path (₹0 cost): All 8 books below have working free PDF links from public GitHub repositories. If one link is slow, try the next mirror. Start with DDIA → Alex Xu Vol.1 → Grokking.
#1 — Must Read First
Designing Data-Intensive Applications
Martin Kleppmann · O'Reilly
The bible of backend system design. Covers databases, replication, partitioning, distributed transactions, and stream processing. Every backend engineer must read this.
📅 Use on: Days 1, 2, 3, 6, 9
PDF Link 1 PDF Link 2
#2 — Core Interview Book
System Design Interview Vol.1
Alex Xu · ByteByteGo
Most interview-focused system design book. URL shortener, rate limiter, notification system, news feed, chat system, YouTube — all with step-by-step diagrams.
📅 Use on: Days 4, 5, 7, 12
PDF Link 1 PDF Link 2 PDF Link 3
#3 — Advanced Systems
System Design Interview Vol.2
Alex Xu & Sahn Lam · ByteByteGo
Advanced problems: proximity service, Google Maps, distributed message queue, metrics monitoring, ad click event aggregation, hotel reservation system.
📅 Use on: Days 6, 11, 12
PDF (GitHub)
#4 — Problem-Solution Format
Grokking the System Design Interview
Design Gurus · Educative
20+ real system design problems with complete step-by-step answers. Great companion for Phase 2 design days — shows worked solutions after you attempt them yourself.
📅 Use on: Days 4, 5, 7, 13
PDF Link 1 PDF Link 2 PDF Link 3
#5 — Advanced Internals
Grokking Advanced System Design
Design Gurus · Educative
Dynamo, Kafka, Cassandra, Chubby, GFS, HDFS internals explained. Shows how real systems at Google and Amazon are actually built under the hood.
📅 Use on: Days 9, 12, 13
PDF (GitHub)
#6 — Architecture Code
Clean Architecture
Robert C. Martin (Uncle Bob)
SOLID principles, component design, dependency rules for backend code. Makes your architecture decisions defensible and shows maturity in interviews.
📅 Use on: Days 8, 10
PDF (GitHub) Chapter Notes
#7 — Microservices Bible
Building Microservices (2nd Ed.)
Sam Newman · O'Reilly
The definitive guide to microservices. Service decomposition, communication patterns, testing, deployment. Essential reading before Day 8.
📅 Use on: Day 8
gmoral/Books repo
#8 — 100% Free Official
Designing Distributed Systems
Brendan Burns · Microsoft (Free)
Official free ebook from a Microsoft Distinguished Engineer. Covers distributed patterns: sidecars, ambassadors, adapters, batch processing, and event-driven design.
📅 Use on: Days 6, 9, 10
